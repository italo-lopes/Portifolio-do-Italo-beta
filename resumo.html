HTML

O html é só um esqueleto do do texto por isso é considerada uma Linguagem de marcação

Saber as tags sematicas é muito importante para descrever o significado do conteudo

O DOCTYPE AVISA AO NAVEGADOR QUE ESSE CONTEUDO É UM ARQUIVO HTML


Conteudo

head -> title : tag que parece na aba toda as informação de cabeça

body -> h1 , p, link tudo que é informação da pagina \(corpo)

CSS

Cascading Style Sheet : Folha em estilo cascata

seletor {propriedade : valor;}

Ex:

p {color : red;}

tags, elemento , atribudo do elemento

JS - JavaScript

Podemos caracterizar como unica linguagem de programaçãao

Fazemos assim deixando o pagina com movimentação

Essa linguagem pode ser usada tanto do lado do servidor quanto do lado do cliente : usado uma framework chamada -Node.js-

Resumo

HTML : Liguagem de marcação que estrutura os elementos

CSS : Linguagem de estilo que define cores e fontes

JSv : linguagem de programção deixa pagina com mais movimento(dinamico) e lidar melhor com dados recebidos e enviados.
